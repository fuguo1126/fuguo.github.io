## 类型
- bool布尔类型:
```
  true|false
```
- Integer 整型 

```
  $a = 123;// 十进制数
  $a = -123; // 负数
  $a = 0123; // 八进制数 (等于十进制 83)
  $a = 0x1A; // 十六进制数 (等于十进制 26)
  $a = 0b11111111; // 二进制数字 (等于十进制 255)
```
- Float 浮点型 
```
  - $a = 1.234;
```
- String 字符串 

```
  单引号|双引号
```
- Array 数组 
```
   array()
     array(  key =>  value
     , ...
     )
```
- Object 对象 
  - 要创建一个新的对象 object，使用 new 语句实例化一个类:1

```
    <?php
    class foo
    {
        function do_foo()
        {
            echo "Doing foo."; 
        }
    }

    $bar = new foo;
    $bar->do_foo();
    ?> 
```
