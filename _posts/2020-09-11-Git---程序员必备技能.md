## 命令行工具 055
- 解析命令（命令=软件名）-> 指定你要的事情，用哪个软件多做
- 只有本地版本库中的代码才能，推送到远程服务仓库（本地版：管理保存本地分支。本库：文件夹）

- master分支->本地默认分支
 - git init 初始化本地管理本地库（git：主命令，init：子命令）
 - git status ：查看本地管理库状态
    - 本地管理库：1.暂存区  2.本地库
- 查看代码是否提交：
  - 如果暂存区都未提交会显示红色
  - 如果提交->暂存区，显示为绿色
  - 如果提交->本地库，显示"......clean"

 - git add . =>(.)当前本地夹中所有内容
 - git commit  -m  ""  （-m:参数）
 - git clone  克隆
 - ls 查看当前文件夹的内容
 - cd 切换
 - git push 推送
    - （当本地版管理库->远程版本库必须状态为clean）(推送代码之前，必须是干净的)
- 标记语言 （#标题 最多6个#）

## 常用的命令
- "git status" 查看本地版本的状态
